## Query-Selector Solo Sprint

> 무엇을 하는 스프린트 인가요?

- html 파일의 원하는 태그를 고르는 방법, 그리고 값을 변경하는 방법을 익혀보는 스프린트 입니다.
- 대표적으로 많이 쓰이는 셀렉터로는 `getElementById`, `getElementsByClassName`, `querySelector`, `querySelectorAll` 등이 있습니다.
- 더 다양한 종류의 셀렉터, 그리고 자세한 설명을 원하신다면 [링크](https://www.w3schools.com/jsref/met_document_queryselector.asp) 를 참조하세요!

> 무엇을 해야 하나요?

- `index.html` 파일에 미리 작성된 페이지가 있습니다.
- `index.html` 을 직접 수정하지 않고, `index.js` 파일에서 셀렉터를 사용해 원하는 값을 얻어내야 합니다.
- `index.js` 파일 상단에 있는 미리 선언해둔 변수들 안에 알맞은 html tag안의 값을 담아보세요!
